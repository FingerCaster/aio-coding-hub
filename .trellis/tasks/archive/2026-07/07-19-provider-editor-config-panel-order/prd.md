# 供应商编辑弹窗配置区顺序微调

## Goal

将流式空闲超时移到账户用量之前，使账户用量、重试覆盖与限流配置连续排列。

## Requirements

- **R1**：只调整供应商编辑弹窗内置配置区的显示顺序，不修改供应商卡片、后端、IPC、查询、绑定或持久化。
- **R2**：将“流式空闲超时覆盖（秒）”移动到“账户用量”面板之前。
- **R3**：移动后，内置高级配置按“账户用量 → 覆盖全局重试策略 → 限流配置”的顺序呈现，中间不再夹着流式超时输入项。
- **R4**：保留流式空闲超时、账户用量、重试覆盖和限流配置的全部字段、校验、展开状态及保存语义；不改变可选插件贡献区的渲染逻辑。
- **R5**：桌面和窄屏下保持现有间距、宽度与响应式行为，不新增视觉样式。
